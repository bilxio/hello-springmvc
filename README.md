hello-springmvc
=======

这是一个关于SpringMVC框架如何使用的基本演示

目的
------
该项目主要是为了展示SpringMVC作为Java新一代MVC开发框架的基本使用，其中包含（不限于）了以下内容

- Spring **3.1.1.RELEASE**
- SpringMVC **3.1.1.RELEASE**
- Hibernate **4.1.0.Final**
- PostgeSQL **9**
- Velocity **1.7**

如何使用
-----
非常简单，

- 首先下载源码，使用Maven打包，部署到`Tomcat`之类的servlet容器
- 然后在你的postgreSQL数据中执行一下init.sql
- 启动服务器，开始你的SpringMVC之旅吧！
